## R2. Hand re-derivation of the ADE / root-count argument

**Setup and Gram identities — verified.** Centered rows `y_S = 1_S-(m/N)1`
give `tQ = YY^T + (2t-R/N)J` with `Q = 2I+A`. With `h^2 = 2-R/(Nt)` and
`alpha_S = t^{-1/2}y_S + h e` one gets exactly `<alpha_S,alpha_T> = Q_ST`
(diagonal 2, off-diagonal in `{0,1}`) — I re-expanded this:
`t^{-1}<y_S,y_T> = Q_ST-2+R/(Nt)` and `+h^2 = +2-R/(Nt)` cancel to `Q_ST`.
The dual vector `zeta = h^{-1}P_V e` gives `<alpha_S,zeta> = 1` and
`||zeta||^2 = h^{-2}||P_V e||^2 <= h^{-2} = Nt/(2Nt-R) = rho(t)` (eq (7)). All
correct. PSD needs `2Nt-R>0`: at `t0` this is `70,500,333,905 > 0`. `h^2>0`
likewise. **PROVED.**

**Modular rank cap — verified.** `1_L = m^{-1}X1_N` puts `col(J) ⊆ col(X)`, so
(3) gives `rank_Fp(Q) <= rank_Fp(X) <= d0` (eq (4)); `rank_R(Q) <= N`. Uses no
modular-to-real identification. **PROVED.**

**Lattice is ADE — verified (standard).** `Lambda = span_Z{alpha_S}` is even,
integral, positive-definite, generated by its norm-2 vectors, hence an
orthogonal sum of `A/D/E` root lattices (Witt). **PROVED (cited standard).**

**Counting Lemma (8), `|S| <= rho r + 4 rho(16-rho)` — mixed.** I verified,
in full:
- *Universal Coxeter bound* `|S∩C| <= H_C Z_C`: from (9) `Σ_{β∈Φ_C}<β,z>^2 =
  2H_C Z_C`; the selected roots and their (unselected, since inner product `-2`
  is barred) negatives contribute `2` each, so `2|S∩C| <= 2H_C Z_C`. Correct,
  type-independent. **PROVED.**
- *`A_s`*: `(s+1)Z_C <= rho·s + Z_C` since `Z_C <= Σ_C Z_C = ||zeta||^2 <= rho`.
  **PROVED.**
- *`E`*: `H<=30`, so `|S∩C| <= 30 Z_C <= rho·s + 30 Z_C`. **PROVED.**
- *Half-integral `D_s`*: `|z_k|>=1/2` gives `s <= 4 Z_C`; a size-`q` matching is
  `q` orthogonal height-1 norm-2 roots, so Bessel `q/2 <= Z_C < 17/2` forces
  `q <= 16`; half-integrality forbids `e_i+e_j` and `e_i-e_j` both selected
  (would force `z_j=0`), so at most one root per support and `|S∩C| <=
  (#edges) <= q·s <= 16s = rho·s + 4(16-rho)Z_C` (the last equality is exactly
  `s <= 4Z_C`). I reconstructed each sub-step. **PROVED.**
- *Summation*: each per-component excess is `<= 4(16-rho)Z_C` because
  `4(16-rho) > 30 > 1` (needs `rho < 17/2`); summing, `Σ excess <=
  4(16-rho)Σ Z_C <= 4(16-rho)rho`. **PROVED.**
- *Integral-coordinate `D_s`* — **OPEN GAP.** The note disposes of this in one
  sentence ("Counting supports meeting a nonzero coordinate, with pairwise
  nonnegativity, gives fewer than `rho·s` selected roots"). I could rigorously
  reduce the *type-1* roots (one endpoint in the `<=8` nonzero coordinates,
  other endpoint a zero coordinate) to `<= 8(s-|K|) < rho·s` by counting per
  zero-coordinate with the sign-consistency forced by pairwise nonnegativity.
  But the note does not address (i) *type-2* roots (both endpoints nonzero) or
  (ii) small `s`, and my own attempts to absorb those into the stated `< rho·s`
  are loose enough that I cannot close them by hand. This matters: see the
  margin note below.

**Reduction to nonexistence — verified.** With (8), `r >= L/rho0 - 64 + 4rho0`
`= 2,029,720 = d0+15`. A rank-`r` symmetric rational matrix has a nonsingular
principal Gram block `B`; `rank_Fp(B) <= d0` and Smith normal form give
`p^{r-d0} | det(B)`, so `det(B) >= p^{15}`. The independent basis roots span an
ADE sublattice with `c <= 16` components (Bessel: `c/2 <= rho0 < 17/2`), so
`det(B) <= Π(s_j+1) <= (N+1)^{16}`. Then `p^{15} <= det(B) <= (N+1)^{16}`, i.e.
`2^{450} < p^{15} <= (N+1)^{16} < 2^{352}` — contradiction. **PROVED** given (8).

**Margin (why the gap is material).** The determinant step needs only
`r-d0 >= 11` (I computed `ceil(16·log2(N+1)/log2 p) = 11`); the note delivers
`15`, margin `4`. Equivalently the Lemma's additive term may be as large as
`~35·rho ≈ 289` before the floor drops to `d0+10`; the claimed `4rho(16-rho) ≈
256` sits `~33` under that. So the *constant* in (8) is load-bearing, and the
unproved integral-`D_s` sub-case cannot simply be waved through — it must be
shown not to erode the additive budget past `~289`.

**Not a counterexample.** No surviving family `F` is exhibited, and none is
expected; the abstract Lemma may well be true with the author's full argument.
This is a proof-*completeness* gap in the note's most compressed step, flagged
because the promotion gate is "before citing outside experimental/".
