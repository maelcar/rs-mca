## Statement

Let `k` be algebraically closed, and let `psi in k[x,x^-1]` be a nonconstant
Laurent polynomial of degree `d` as a map `P^1 -> P^1`.  Assume
`char(k)=0` or `char(k)=p>d`, so `psi` is tame and separable.

If the reduced non-diagonal fiber product

```text
psi(x) = psi(y),        (x,y) in G_m^2
```

has a geometric toral component, then exactly one of the following charged
forms occurs, after scaling `x`:

```text
cyclic:    psi(x) = F(x^m),                    m > 1, F in k[z,z^-1],
dihedral:  psi(x) = F(x^m + alpha x^-m),       m >= 1, alpha in k^*, F in k[z].
```

Conversely, either displayed form produces the corresponding toral components
of the fiber product.

There are two wording corrections relative to the older informal statement.
In the cyclic Laurent branch the outer map is Laurent, not necessarily
polynomial; for example `x^m + x^-2m = F(x^m)` with `F(z)=z+z^-2`.
If `psi` is allowed to be a general rational function rather than Laurent,
the corresponding outer map is rational; for example
`R(z)=z+1/(z^2+1)` gives a cyclicly invariant rational map `R(x^m)` that is
not Laurent.

## Proof

A toral component is a translate of a one-dimensional subtorus, so after
choosing a primitive equation it has the form

```text
x^a y^b = c,        gcd(a,b)=1.
```

Normalize this curve by `s in G_m`.  Then `x` and `y` become monomials in `s`
with exponents `|b|` and `|a|`.  Since `psi(x(s))=psi(y(s))` as maps from
`P^1`, and `psi` has positive degree, degree multiplicativity gives
`|a|=|b|`.  Primitivity forces `|a|=|b|=1`.  Hence every primitive toral
component is the graph of a tame automorphism of `G_m`:

```text
y = lambda x        or        y = lambda / x.
```

Let

```text
G = { tau in Aut(G_m) : psi(tau x) = psi(x) }.
```

Because `p>d`, the map is separable, so a generic fiber has `d` distinct
points and `G` acts faithfully on it.  Thus `G` is finite.  Finite subgroups
of

```text
Aut(G_m) = k^* semidirect {x -> x, x -> x^-1}
```

are cyclic rotation groups or dihedral groups.

If `G` is cyclic of order `m`, scale so it is generated by `x -> zeta x`.
Then the invariant field is `k(x)^G = k(x^m)`.  Since `psi` is fixed by `G`,
there is `F in k(z)` with `psi=F(x^m)`.  A Laurent polynomial has poles only
at `0` and `infinity`; therefore `F` can have poles only at `0` and
`infinity`, so `F in k[z,z^-1]`.  If `psi` is an ordinary polynomial, the
pole at `0` is also absent, hence `F in k[z]`.

If `G` is dihedral, its rotation subgroup has order `m`; after scaling the
reflection has the form `x -> alpha^(1/m)/x`, and the invariant field is

```text
k(x)^G = k(x^m + alpha x^-m).
```

Again `psi=F(x^m + alpha x^-m)` for some rational `F`.  The map
`z=x^m+alpha x^-m` has finite nonzero fibers over every finite value.  If
`F` had a finite pole, then `F(z(x))` would have a finite nonzero pole, which
a Laurent polynomial cannot have.  Thus `F` has only a pole at infinity, so
`F in k[z]`.

The converse is direct.  If `psi=F(x^m)`, then `psi(zeta x)=psi(x)` for every
`m`th root of unity `zeta`.  If `psi=F(x^m+alpha x^-m)`, the same rotations
and the reflection `x^m -> alpha x^-m` preserve `psi`, producing the cyclic
and dihedral toral components.
