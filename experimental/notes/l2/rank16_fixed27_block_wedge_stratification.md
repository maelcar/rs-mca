## Exact theorem

Assume `2 <= r <= 6`.  Put

```text
H_0 = h + g V_*,
H_j = g E_j,  1 <= j <= r.
```

Every polynomial of degree below `3B` has a unique block expansion

```text
Q(X) = Q_0(X) + X^B Q_1(X) + X^(2B) Q_2(X),
deg Q_k < B.
```

Write its block polynomial as

```text
mathcal B(Q)(T) = Q_0 + T Q_1 + T^2 Q_2
```

and set `M_j(T)=mathcal B(H_j)(T)`.  Finally put

```text
Omega_r(T) = M_0(T) wedge ... wedge M_r(T),
L_S(T) = product_(i=1)^7 (T-y_i).
```

Then:

1. For every `y in mu_64`, the direction columns
   `M_1(y),...,M_r(y)` are independent.  At `y_i`, the full matrix has
   one-dimensional kernel generated by `(1,c_i1,...,c_ir)`.

2. Exactly one of the following branches occurs.

   **Transverse branch.** If `Omega_r` is nonzero, then

   ```text
   Omega_r(T) = L_S(T) Theta_r(T),
   deg_T Theta_r <= 2r-5.
   ```

   Hence the transverse degrees for `r=3,4,5,6` are at most `1,3,5,7`.
   The transverse branch is impossible for `r=2`.

   **Syzygetic branch.** If `Omega_r=0`, there is a primitive polynomial
   vector `(a_0,...,a_r)`, unique up to a nonzero scalar, such that

   ```text
   sum_(j=0)^r a_j(T) M_j(T) = 0,
   deg a_j <= 2r.
   ```

   Its first coordinate satisfies

   ```text
   a_0(y) != 0 for every y in mu_64,
   g(X) divides a_0(X^B).
   ```

   If `e=deg a_0` and `a_0(X^B)=g(X)s(X)`, then

   ```text
   3 <= e <= 2r,
   deg s = eB-a.
   ```

   At the seven source labels the correct kernel coordinates are

   ```text
   c_ij = a_j(y_i) / a_0(y_i).                            (2)
   ```

   Writing `rho_i=P_i/(X^B-y_i)=q_i^(-1)R_i`, there are polynomials
   `b_ij(T)` of degree at most `e-1` such that

   ```text
   s(X) rho_i(X)
     = sum_(j=1)^r b_ij(X^B) E_j(X).                      (3)
   ```

   Moreover every `z` with `a_0(z)!=0` gives a divisibility identity

   ```text
   X^B-z divides
     H_0 + sum_(j=1)^r (a_j(z)/a_0(z)) H_j.               (4)
   ```

3. In affine rank two the syzygetic branch is forced and `e` is either
   three or four.

   **Cubic case.** If `e=3`, then `deg s=30832`.  There is a fixed space
   `W_3` of dimension at most three such that

   ```text
   rho_i in W_3 + X^B W_3  for every i.
   ```

   **Quartic case.** If `e=4`, then `deg s=63600`.  There is a fixed space
   `W_2` of dimension at most two such that

   ```text
   rho_i in W_2 + X^B W_2  for every i.
   ```

In particular, define the root-free `B`-imprimitivity degree of `g` by

```text
iota_B(g) = min deg A,
```

where the minimum is over nonzero `A in F_p[T]` such that
`g | A(X^B)` and `A(y)!=0` for every `y in mu_64`.  Then a rank-two
seven-label family forces

```text
iota_B(g) in {3,4}.
```

Thus every source cell with `iota_B(g)>=5` has no rank-two seven-label
family.

## Proof

### 1. The block map is the full-fibre remainder map

For `Q=Q_0+X^B Q_1+X^(2B)Q_2`, reduction modulo `X^B-y` gives

```text
Q mod (X^B-y) = Q_0 + y Q_1 + y^2 Q_2 = mathcal B(Q)(y). (5)
```

Since `D<3B`, every `M_j` has `T`-degree at most two.  Equation (1) and
the affine expansion give

```text
M_0(y_i) + sum_j c_ij M_j(y_i) = 0.                      (6)
```

### 2. Direction rank

Suppose `sum_j lambda_j M_j(y)=0` for some `y in mu_64`.  By (5),
`X^B-y` divides `g sum_j lambda_j E_j`.  The factor `X^B-y` splits over
`H`, while `g` is root-free there, so it divides `sum_j lambda_j E_j`.
That polynomial has degree below `B`; hence it is zero, and independence of
the `E_j` gives every `lambda_j=0`.  This proves the first assertion.

### 3. Wedge interpolation

Every coefficient of `Omega_r` is an `(r+1)`-minor of columns of degree at
most two, so its degree is at most `2(r+1)`.  Equation (6) makes it vanish
at the seven distinct `y_i`.  If it is nonzero, componentwise division by
`L_S` yields the transverse branch and the bound `2(r+1)-7=2r-5`.  For
`r=2`, the degree is at most six, so seven roots force `Omega_2=0`.

### 4. Primitive syzygy

Assume `Omega_r=0`.  Over `F_p(T)`, the direction columns have rank `r`
and the complete matrix has rank `r`; its kernel is one-dimensional.  Cramer
minors give a polynomial kernel vector with coordinate degrees at most `2r`.
Dividing by the coordinate gcd produces a primitive generator
`a=(a_0,...,a_r)`, unique up to a unit.

The substitution `T=X^B` is injective, so the block syzygy is equivalent to

```text
a_0(X^B) H_0 + g sum_(j=1)^r a_j(X^B) E_j = 0.          (7)
```

If `a_0(y)=0` for a label in `mu_64`, specialization of the block relation
and direction independence make every `a_j(y)` zero.  Then `T-y` divides
all coordinates, contradicting primitivity.  Thus `a_0` is nonzero on all
64 labels.

The anchor `H_0` is an actual normalized locator, so it splits over `H`.
Since `g` is root-free on `H`, `gcd(g,H_0)=1`.  Equation (7) therefore gives
`g | a_0(X^B)`.  As `deg g=a>2B`, this forces `e>=3`; Cramer's bound gives
`e<=2r`.

At `y_i`, both `(1,c_i1,...,c_ir)` and
`(a_0(y_i),...,a_r(y_i))` generate the one-dimensional kernel.  Division by
the proved-nonzero `a_0(y_i)` gives (2).  This orientation is load-bearing:
the reciprocal formula is false when some `a_j(y_i)` vanishes.

Multiply `P_i=H_0+sum_j c_ij H_j` by `a_0(X^B)`, use (7), and cancel `g`:

```text
s P_i = sum_j (c_ij a_0(X^B)-a_j(X^B)) E_j.
```

By (2), every coefficient is divisible by `T-y_i`.  Cancelling the common
factor `X^B-y_i` gives (3).  A block-degree comparison shows
`deg b_ij<=e-1`: a term of `T`-degree at least `e` would create a nonzero
block of degree at least `eB`, while the left side has degree below `eB`.
Evaluating the block syzygy at an arbitrary `z` with `a_0(z)!=0` proves (4).

### 5. Pluecker content and rank-two rigidity

Let

```text
Pi_r(T) = M_1(T) wedge ... wedge M_r(T).
```

Every cofactor syzygy is a polynomial multiple of the primitive vector.
Therefore

```text
Pi_r(T) = a_0(T) Xi_r(T),
deg_T Xi_r <= 2r-e.                                      (8)
```

For `r=2`, `e` is three or four.  If `e=4`, `Xi_2` is a nonzero constant,
so all specialized direction planes equal one fixed two-space `W_2`.  If
`e=3`, `Xi_2` is at most linear.  The Pluecker quadrics vanish identically
on it, so its projective image is a line in `Gr(2,U)`.  Such a line consists
of two-planes containing one fixed line and contained in one fixed
three-space `W_3`; the constant-plane case is included.

The global syzygy places `M_0(T)` in the direction plane over `F_p(T)`, so
all coefficient blocks lie in the same `W_2` or `W_3`.  Each `P_i` is a
quadratic block polynomial vanishing at `T=y_i`; division by `T-y_i` leaves
a linear block polynomial.  This gives the two displayed residual-space
conclusions and completes the proof.

## Source compiler

Before fixed-core cancellation, write the full locator as `L_i=G_C A_i`,
where

```text
G_C(X) = product_(z in C) (X^B-z),
L_i == q_i h_src (mod g).
```

Every root of `G_C` lies in `H`, while `g` is root-free there, so `G_C` is
invertible modulo `g`.  Defining

```text

```

gives `A_i == q_i h (mod g)`, hence (1), with
`deg W_i<=D-a=28897`.  The theorem therefore applies to the literal source
cell, not to an abstract family of supports.

The proof does not use the remaining live residual restrictions, but any
candidate surviving the theorem must still preserve them: each `R_i` must be
monic, squarefree, degree `63601`, split over `H`, avoid all selected fibres,
have no additional complete `q64` fibre, have residual footprint at least
four, and contain at most thirteen complete `q32` error pairs.
