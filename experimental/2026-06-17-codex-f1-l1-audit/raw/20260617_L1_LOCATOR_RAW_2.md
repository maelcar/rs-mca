## ROUTE_CUT / COUNTEREXAMPLE

**Formal status tag: COUNTEREXAMPLE** to the quotient-only L1 proof route.
This is **not** a counterexample to `conj:final-locator`; it is a finite counterexample to the tempting mechanism “after quotient removal, monomial-prefix collisions disappear.” The remaining L1 theorem still needs a genuine finite-field aperiodic local-limit estimate.

## 2. Restated L1 local-limit target with field and ledger separation

Let

[
B=\mathbb F_{q_{\rm gen}},\qquad H_n\le B^\times,\qquad |H_n|=n=2^m,
]

where `H_n` is generated-field smooth: the field generated by the domain and locator coefficients is exactly `B`, so the entropy denominator is `q_gen`, not an ambient extension size.

Let

[
F\supseteq B
]

be an optional ambient or protocol line field. For bare L1, take `F=B`. If a protocol uses `F>B`, that is not free L1 credit: Paper C moves the list problem into interleaving and records the extension degree separately.

Let

[
k=\rho n+O(1),\qquad a=k+\sigma,\qquad \eta=\sigma/n,
]

and

[
\delta=1-a/n=1-k/n-\sigma/n.
]

If `k=ρn` exactly, then

[
\delta=1-\rho-\eta.
]

The L1 target is:

[
|\Fib_U(a)|\le n^{B_L}
]

for every received word/interpolant `U`, whenever

[
\sigma\ge C_{\rho,B_L,\varepsilon}\frac n{\log_2 n},
]

[
\sigma\log_2 q_{\rm gen}\ge (1+\varepsilon)\log_2\binom n{k+\sigma},
]

and all active quotient-core contributions are absent, explicitly removed, or bounded by the quotient budget.

For the list quotient-core profile, use:

[
K\le H,\qquad |K|=M,\qquad N=n/M.
]

The Paper B quotient-core obstruction is active when

[
M\mid \gcd(n,k),\qquad \sigma<M,\qquad k/M\le N-1,
]

and contributes

[
\binom{N-1}{k/M}.
]

The MCA slack variable `t` is not the same as the list slack `σ`. In L1 the slack is `σ=a-k`. Paper B’s quotient-MCA sections use a local quotient slack `t`; it should not be converted into `σ` without a separate theorem.

For arities, bare L1 is the base-list statement with

[
\mu=1,\qquad \nu=1.
]

Paper C later uses a protocol list arity `μ` through an interleaved list bound, often the trivial product

[
\widehat L_\mu\le \widehat L_1^\mu,
]

while `ν` is the implementation interleaving factor on the MCA side. Neither `μ` nor `ν` changes the generated-field entropy denominator.

## 3. Narrow-case result: finite quotient-separated monomial-prefix collision

This is the route cut.

Take

[
B=F=\mathbb F_{17},\qquad q_{\rm gen}=17,\qquad H=\mathbb F_{17}^{\times}.
]

Then

[
n=16,\qquad k=6,\qquad \rho=3/8,\qquad \sigma=4,\qquad a=s=k+\sigma=10.
]

So

[
\eta=\sigma/n=1/4,\qquad \delta=1-s/n=6/16=3/8.
]

The generated-field entropy ledger is already positive:

[
4\log_2(17)-\log_2\binom{16}{10}
=4\log_2(17)-\log_2(8008)
\approx 3.3826>0.
]

The exact quotient-core profile is empty:

[
\gcd(n,k)=\gcd(16,6)=2.
]

The only nontrivial divisor of `gcd(n,k)` is `M=2`, but active quotient cores require `σ<M`; here `4<2` is false. Thus no Paper B list quotient-core contribution is active.

Now define two `10`-subsets of `H`:

[
S={1,2,3,4,5,6,7,9,10,12},
]

[
T={1,2,3,8,10,11,13,14,15,16}.
]

A direct calculation in `F_17` gives

[
(e_1(S),e_2(S),e_3(S),e_4(S))
=============================

# (e_1(T),e_2(T),e_3(T),e_4(T))

(8,12,13,7).
]

Thus `S` and `T` lie in the same monomial-prefix fiber

[
\Phi_4^{-1}(8,12,13,7).
]

Equivalently, for

[
U_c(X)=X^{10}-8X^9+12X^8-13X^7+7X^6
\in\mathbb F_{17}[X],
]

or

[
U_c(X)=X^{10}+9X^9+12X^8+4X^7+7X^6,
]

both

[
P_S=U_c-L_S,\qquad P_T=U_c-L_T
]

have degree `<6`, hence are RS codewords agreeing with `U_c` on `S` and `T`, respectively, by `prop:monomial-fiber`.

This collision is not quotient-periodic after quotient removal. The symmetric difference is

[
S\triangle T={4,5,6,7,8,9,11,12,13,14,15,16}.
]

For `σ=4`, quotient-periodic exceptions in the `thm:no-collision` sense would require a subgroup order `M>4`, so only `M=8` or `M=16` can occur. With generator `3` of `F_17^\times`, the two cosets for `M=8` are

[
{1,2,4,8,9,13,15,16},
]

[
{3,5,6,7,10,11,12,14}.
]

The symmetric difference intersects each in exactly six elements, so it is not a union of `M=8` cosets. It is also not the whole group, so it is not an `M=16` coset union.

Therefore:

[
\Phi_4(S)=\Phi_4(T),
]

entropy clears,

[
\mathcal Q_H(a,k)=-\infty,
]

but the collision is genuinely aperiodic.

## 4. What this cuts

The following route is false:

> Prove L1 in polynomial generated fields by removing quotient cores and then showing that all remaining monomial-prefix finite-field collisions disappear.

They do not disappear. The `F_17` example above has:

[
q_{\rm gen}=17,\quad n=16,\quad k=6,\quad \sigma=4,\quad a=10,
]

positive entropy margin, empty quotient-core profile, and a non-quotient monomial-prefix collision.

This does not disprove the desired bound

[
|\Phi_\sigma^{-1}(c)|\le n^{B_L}.
]

The displayed fiber has size at least `2`, not superpolynomial size. But it proves that the local-limit theorem cannot be a no-collision theorem in polynomial fields. It must be a multiplicity bound for aperiodic finite-field collisions.

This also matches `rem:galois-limit`: the existing Galois/norm route reaches only the quasi-polynomial split-prime range. At the critical scale

[
\sigma\asymp n/\log n,
]

`thm:no-collision` requires roughly

[
p>\exp!\left(\Theta\left(\frac{n\log n}{\sigma}\right)\right)
=============================================================

\exp(\Theta((\log n)^2)),
]

whereas polynomial generated fields have only

[
p=\exp(O(\log n)).
]

So the current norm-divisibility proof cannot bridge the missing factor of `log n`.

## 5. Verification pseudocode

```python
from math import comb, log2, gcd

p = 17
H = set(range(1, 17))
n = 16
k = 6
sigma = 4
s = k + sigma

S = {1,2,3,4,5,6,7,9,10,12}
T = {1,2,3,8,10,11,13,14,15,16}

def elem_sym(A, r, p):
    e = [0] * (r + 1)
    e[0] = 1
    for x in A:
        for j in range(r, 0, -1):
            e[j] = (e[j] + e[j-1] * x) % p
    return tuple(e[1:])

assert len(S) == s
assert len(T) == s
assert elem_sym(S, sigma, p) == (8, 12, 13, 7)
assert elem_sym(T, sigma, p) == (8, 12, 13, 7)

# Generated-field entropy ledger.
entropy_margin = sigma * log2(p) - log2(comb(n, s))
assert entropy_margin > 0

# Quotient-core profile is empty.
active_M = []
for M in [1, 2, 4, 8, 16]:
    if gcd(n, k) % M == 0 and M > 1 and sigma < M and k // M <= n // M - 1:
        active_M.append(M)
assert active_M == []

# Quotient-periodic check for M > sigma.
g = 3  # primitive generator of F_17^*
D = S ^ T

def cosets_of_order(M):
    step = n // M
    return [
        {pow(g, r + t * step, p) for t in range(M)}
        for r in range(step)
    ]

def is_union_of_cosets(D, M):
    for C in cosets_of_order(M):
        if D & C and not C <= D:
            return False
    return True

assert not is_union_of_cosets(D, 8)
assert not is_union_of_cosets(D, 16)
```

## 6. EXACT_NEW_WALL

The first irreducible missing lemma is not a quotient-classification lemma. It is this finite-field aperiodic multiplicity bound:

**Missing lemma, minimal monomial-prefix form.**
Fix `ρ∈(0,1)`, `B_L>0`, and `ε>0`. There is `C=C(ρ,B_L,ε)` such that for every generated-field smooth domain

[
H\le \mathbb F_q^\times,\qquad |H|=n=2^m,\qquad q=q_{\rm gen}=\operatorname{poly}(n),
]

with

[
k=\rho n+O(1),\qquad a=k+\sigma,
]

if

[
\sigma\ge Cn/\log_2 n,
]

[
\sigma\log_2 q\ge(1+\varepsilon)\log_2\binom n a,
]

and the quotient-core profile is absent or paid, then for every `c∈F_q^σ`,

[
#{S\in \binom H a:\Phi_\sigma(S)=c
\text{ and }S\text{ is not charged to an active quotient-periodic class}}
\le n^{B_L}.
]

The `F_17` construction proves that this lemma must allow isolated aperiodic collisions. It cannot assert aperiodic injectivity.

The arbitrary-word version needed for `conj:arbitrary-local` is strictly harder: replace `Φ_σ(S)=c` by

[
\deg(U\bmod L_S)<k
]

uniformly over every interpolant `U∈F_q[X]_{<n}`. No theorem in the current repository supplies that bound.

## 7. Paper B plug-in consequence

A proof of the missing aperiodic multiplicity lemma would plug into:

[
\texttt{conj:prefix-local}
\Rightarrow
\text{monomial-prefix half of }\texttt{thm:conditional-list}.
]

A further arbitrary-`U` version would plug into:

[
\texttt{conj:arbitrary-local}
\Rightarrow
\texttt{thm:conditional-list}
\Rightarrow
\texttt{ass:locator} \text{ in Paper C’s certificate}.
]

The route cut above says the plug-in cannot be obtained by quotient removal plus no-collision. The remaining theorem must be a true polynomial-field local-limit estimate for aperiodic finite-field locator fibers.
