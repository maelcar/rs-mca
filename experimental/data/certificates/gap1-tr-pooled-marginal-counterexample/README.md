# GAP-1 pooled-marginal TR counterexample certificate

**Author:** Manuel E. Rey-Álvarez Zafiria

The canonical certificate is regenerated by:

```bash
python experimental/scripts/verify_gap1_tr_pooled_marginal_counterexample.py
```

The verifier is deterministic and uses only the Python standard library.  It
checks the exact `F_337` enumeration, ten exact asymptotic arithmetic rows, the
literal terminal-reserve source statement, and the current DAG statuses.

Expected final line:

```text
PASS_WITH_TWO_COSET_MARGINAL_TR_COUNTEREXAMPLE
```
