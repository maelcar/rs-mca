# E25 Dihedral Toy Audit Certificate

Deterministic certificate for the E25 toy dihedral audit.

Replay from the repository root:

```bash
python3 experimental/scripts/verify_e25_dihedral_toy_audit.py --write
python3 experimental/scripts/verify_e25_dihedral_toy_audit.py --check
```

The JSON records the `F_17` and `F_97` `mu_16` locator audit, the
Chebyshev-character basis-line alignment counts, and the zero unpaid residual
after tangent, cyclic-quotient, and dihedral/antipodal charges.
