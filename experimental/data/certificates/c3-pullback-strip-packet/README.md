# C-3 Pullback-Strip Packet Certificate

Deterministic certificate for the C-3 pullback-strip packet.

Replay from the repository root:

```bash
python3 experimental/scripts/verify_c3_pullback_strip_packet.py --write
python3 experimental/scripts/verify_c3_pullback_strip_packet.py --check
```

The JSON records the finite `pma_pullback_lists` replay, the primitive
residual split, and the E15 challenger classification.
