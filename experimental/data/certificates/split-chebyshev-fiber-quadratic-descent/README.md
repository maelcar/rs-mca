# Split Chebyshev fibre quadratic-descent certificate

**Author:** Manuel E. Rey-Álvarez Zafiria

From the repository root run

```text
python experimental/scripts/verify_split_chebyshev_fiber_quadratic_descent.py
```

The program uses only the Python standard library.  It writes no files and
prints the exact JSON payload recorded in `verification_output.json`.
`SHA256SUMS.txt` authenticates the note, verifier, and recorded output.
