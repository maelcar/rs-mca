# xr-triangle-eliminant-form certificate

This directory pins the toy linear-algebra recomputation for DAG node
`xr_triangle_eliminant_form` (2c-beta-3a).

Run:

```bash
python3 experimental/scripts/verify_xr_triangle_eliminant_form.py
```

The verifier compares the chart normal form, the twisted form, and brute
stacked alignment rank on the rows recorded in
`toy_triangle_eliminant.json`.
